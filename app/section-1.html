Grid & Layout

Create powerful multi-device layouts quickly and easily with the default 12-column, nestable grid. If you're familiar with grid systems, you'll feel right at home. If not, you'll learn quickly.

This is a twelve column section in a row. Each of these includes a div.callout element so you can see where the columns are - it's not required at all for the grid.
